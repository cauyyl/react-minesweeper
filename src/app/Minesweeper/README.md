## 开发步骤
1. 实现扫雷布局UI
2. 创建地图矩阵，并随机生成地雷
3. 增加坐标指示，实现坐标输入功能
4. 实现通过坐标扫雷
5. 如果命中安全区，UI变化，如果命中雷区，游戏结束

## 地雷矩阵数据
1. 随机生成地雷
2. 数据结构设计，存储是否地雷，存储是否被“扫”过

## 如何随机生成地雷
1. 根据宽度和高度计算出总格子数，生成格子数组，初始化为0（安全区）
2. 根据雷数随机生成地雷位置，修改格子数组，标记为1（雷区）
3. 把格子数组转化成格子矩阵

## 交互方式
1. 点击
2. 键盘输入坐标
